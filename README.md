# ENVS-193DS_infographic_spring-2026

## General information

This repo provides the code for visualizations of data collected as part of a quarter-long personal project.

## Data and file information

There is a `data` folder in which data files are stored.

There is a `code` folder in which all code for data visualizations are stored.

There is an `images` folder in which images used in the visualizations are stored.

There is an `exports` folder in which figures exported using ggsave are stored.
